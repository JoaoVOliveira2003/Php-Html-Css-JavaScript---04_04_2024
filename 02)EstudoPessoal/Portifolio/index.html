Projetos


Plataforma de criação de recibos
Este é um software que permite a criação de recibos, os quais podem ser posteriormente baixados em formato PDF. Ele foi desenvolvido para revolucionar o processo de emissão de recibos na Assoalhos Modelo.
Até 2019, os recibos eram feitos manualmente à mão. No entanto, a partir desse ano, fizemos a transição para o uso de computadores, embora ainda fossem criados individualmente por meio da ferramenta Canvas. Como a criação de recibos via Canvas era trabalhosa, decidi inovar, desenvolvendo um sistema que utilizasse caixas e permitisse o download dos recibos em formato PDF.
Embora ainda haja a necessidade de alguns pequenos ajustes na funcionalidade de download dos recibos, o sistema já está em uso. Foram utilizadas tecnologias como CSS, JavaScript e HTML no desenvolvimento deste projeto. Este é um comentário para o link no Github
