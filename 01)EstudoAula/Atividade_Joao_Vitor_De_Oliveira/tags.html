Tag "A" - é a tag âncora que é usada para linkar páginas em HTML

Tag "P" - é a tag usada para definir um parágrafo

Tag "H1 ... H2" - são uma série de tags que fazem títulos, sendo o H1 o maior e quando você vai passando, o título vai diminuindo

Tag "IMG" - usada para inserir imagens

Tag "BR" - usada para fazer quebras de linha em um texto

Tag "HR" - cria uma linha horizontal

Tag "I" - deixa o texto em itálico

Tag "DEL" - usada para marcar textos já deletados

Tag "INS" - mostra textos inseridos em versões mais recentes

Tag "SUB" - insere texto como subscrito

Tag "Q" - usada para marcar citações curtas

Tag "ADDRESS" - define informações de contato ou detalhes de endereço

Tag "BDO" - usada para controlar a direção de escrita de textos que são lidos da direita para a esquerda
